### Q1. 다음 프로토타입 기반 객체를 클래스 기반 객체로 바꾸어 주세요.(수영)

```ts
function Person(first, last) {
  this.first = first;
  this.last = last;
}

Person.prototype.getName = function () {
  return this.first + " " + this.last;
};

const marie = new Person("Marie", "Curie");
const personName = marie.getName();
```
