# 아이템61 의존성 관계에 따라 모듈 단위로 전환하기

### Q1.다음 자바스크립트 코드를 타입스크립트로 마이그레이션 시에 오류와 해결책을 말해주세요.(수영)

```ts
class Greeting {
  constructor(name) {
    this.greeting = "Hello";
    this.name = "name";
  }

  greet() {
    return this.greeting + " " + this.name;
  }
}
```
