# 아이템14 타입 연산과 제너릭 사용으로 반복 줄이기

### Q1. A, B 는 각각 어떻게 출력될까요? (경민)

```js
interface SaveButton {
  type: "save";
}

interface LoadButton {
  type: "load";
}

type Button = SaveButton | LoadButton;
type A = Button["type"];
type B = Pick<Button, "type">;

console.log("A = ", A);
console.log("B = ", B);
```
