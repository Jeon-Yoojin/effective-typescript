# 아이템24 일관성 있는 별칭 사용하기

### Q1. 다음은 사용자의 정보를 관리하는 코드입니다. profile.age의 값은 무엇이고 어떤 문제가 발생할 수 있나요? (경민)

```ts
const person = { name: "Alice", age: 25 };

const user = person;
const profile = person;

function updateUserAge(user: { age: number }) {
  user.age += 1;
}

updateUserAge(user);
console.log(profile.age);
```
